======================================
Nextclade: analysis of viral sequences
======================================

Nextclade is an open-source project for viral genome alignment, clade assignment, mutation calling, and quality checks.
It is available as a web application at `clades.nextstrain.org <https://clades.nextstrain.org>`_, as well as a command-line tool.

Nextclade is a part of `Nextstrain <https://nextstrain.org>`_, an open-source project to harness the scientific and public health potential of pathogen genome data. All source code is available on `GitHub <https://github.com/nextstrain/nextclade>`_.
